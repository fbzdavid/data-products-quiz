Data Classification help

Usage

This webapplication will allow you to update a csv file, and do a classification of the data.

Step by step guide

  1. Please check if the data file has a header
  2. Please select the separator for the file
  3. Please select the classification method: None for data plotting
  4. If the method selected in k means, please select teh number of classes
  5. Select the file to upload by clicking the "choose file" button
